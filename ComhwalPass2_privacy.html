개인정보 처리방침 — 컴활패스 2급

시행일: 2026년 5월 5일 · 앱 이름: 컴활패스 2급 · 번들 ID: com.leeminyong.ComHwal

본 방침은 컴활패스 2급 모바일 앱(이하 "앱")이 사용자의 정보를 어떻게 수집·이용·보관·삭제하는지를 안내합니다.

한 줄 요약 — 이 앱은 사용자 정보를 외부 서버로 전송하지 않으며, 모든 학습 기록은 사용자 본인의 기기 안에만 저장됩니다.

1. 수집·저장하는 정보

항목저장 위치목적
풀이 기록 (문제 ID, 정답 여부, 일시, 과목·카테고리·난이도)기기 내부 SwiftData통계·약점 분석
즐겨찾기한 문제·핵심노트 ID기기 내부 SwiftData즐겨찾기 모음
모의고사 기록 (점수·시간·합격 여부)기기 내부 SwiftData모의고사 이력
진행 중인 풀이 상태 (이어 풀기)기기 내부 UserDefaults풀이 재개
설정값 (테마·시험일·알림 시각 등)기기 내부 UserDefaults앱 환경 설정

위 정보는 모두 사용자의 기기를 떠나지 않습니다. 앱은 자체 서버를 운영하지 않으며 외부로 전송하는 통신이 없습니다.

2. 수집하지 않는 정보

  • 이름·전화번호·이메일·주소 등 신원 정보
  • 위치 정보(GPS·Wi-Fi·셀룰러)
  • 연락처·캘린더·사진·미디어 라이브러리
  • 기기 식별자 (IDFA, IDFV 등)
  • 사용 통계·분석·크래시 리포트 (3rd-party SDK 미사용)

3. 권한 사용

권한용도거부 시 영향
알림설정한 시각의 학습 리마인더알림이 표시되지 않음

4. Apple Intelligence (온디바이스 AI)

[AI 코칭] 기능은 iOS 26 이상의 Apple Intelligence (Foundation Models) 를 사용합니다. 모든 처리는 사용자 기기 안에서 이뤄지며 외부 서버로 전송되지 않습니다. 미지원 기기에서는 자동으로 규칙 기반 분석으로 대체됩니다.

5. 음성 출력(TTS)

iOS 시스템 음성 합성(AVSpeechSynthesizer)을 사용하며 모든 처리는 기기 내부에서 이뤄집니다.

6. 제3자 제공·공유

앱은 사용자 정보를 제3자에게 제공·공유·판매하지 않습니다. 제3자 분석·광고 SDK 도 포함되어 있지 않습니다.

7. 보관 및 삭제

  • 모든 학습 기록은 사용자가 앱을 삭제할 때까지 기기에 보관
  • [앱 → 설정 → 기록 초기화] 로 언제든 모두 삭제 가능
  • 앱 삭제 시 데이터는 iOS 정책에 따라 즉시 제거

8. 14세 미만 사용자

앱은 일반 학습 도구로 식별 정보를 수집하지 않습니다. 14세 미만은 보호자 동의 하에 사용해 주세요.

9. 사용자 권리

  • 열람: 모든 데이터를 앱 화면에서 직접 확인
  • 삭제: [기록 초기화] 또는 앱 삭제로 즉시 삭제

10. 보안

모든 데이터는 사용자 기기 내부에 저장되며 iOS 의 샌드박스·암호화 정책의 보호를 받습니다.

11. 처리방침 변경

변경 시 시행일과 함께 본 페이지에 게시합니다.
